# 企业微信机器人通知插件 (WXWork Notification Plugin)

Jenkins 插件：在 Pipeline 和 FreeStyle 项目中发送企业微信机器人消息。

## 文档站

- 在线文档（GitHub Pages）：`https://jenkinsci.github.io/wxwork-notification-plugin/`
- 本地开发文档：
  - `pnpm install`
  - `pnpm docs:dev`
  - `pnpm docs:build`

## 主要能力

- 支持 `text`、`markdown`、`markdown_v2`、`image` 消息
- 支持 `@构建执行人`、`@所有人`、指定手机号
- 同时支持 FreeStyle 与 Pipeline
