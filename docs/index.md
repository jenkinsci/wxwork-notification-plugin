# 企业微信机器人通知插件

Jenkins 插件 `wxwork-notification-plugin`，用于在 FreeStyle 和 Pipeline 项目中发送企业微信机器人通知。

- [安装](/guide/install)

- [Pipeline 项目](/guide/pipeline)
- [常见问题](/guide/faq)
- [企业微信机器人配置（附录）](/guide/settings)

## 功能特性

- 同时支持 FreeStyle 和 Pipeline 项目
- 支持 `text`、`markdown`、`markdown_v2`、`image` 消息类型
- 支持 `@构建执行人`、`@所有人`、指定手机号
- 消息内容支持 Jenkins 环境变量和构建参数
- 支持多机器人配置
