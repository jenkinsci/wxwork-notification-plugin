# 企业微信机器人配置

本页说明如何在 Jenkins 中配置企业微信机器人与成员手机号。

## 一、配置企业微信机器人

需要在 Jenkins 系统配置中维护机器人信息，支持配置多个机器人。

操作步骤：

1. 进入 `管理 Jenkins -> 系统`。
2. 找到 `企业微信机器人` 区域并点击 `新增`。
3. 填写机器人 ID、名称和 Webhook URL。
4. 先进行连接测试，再保存。

## 二、配置企业微信成员手机号

如果需要使用 `@构建执行人`，需为 Jenkins 用户配置企业微信手机号：

1. 打开用户主页，进入 `配置`。
2. 在企业微信手机号字段中填写号码并保存。

注意：

- 手机号必须和企业微信成员绑定号码一致。
- 否则插件无法准确执行 `@构建执行人`。
